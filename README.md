# Universal CRUD Hub

A premium, high-performance CRUD application built with React and Vite. This project features a sophisticated **Glassmorphism UI** and modular architecture.

## 🚀 Key Features

- **Full CRUD Operations**: Create, Read, Update, and Delete items.
- **Glassmorphism UI**: High-end translucent cards with blur effects.
- **Responsive Layout**: Designed to look stunning on all screen sizes.
- **Modern Tech Stack**: React (Hooks), Vite, and Vanilla CSS Grid/Flexbox.
- **Animations**: Silky smooth fade-in animations for a premium user experience.

## 🛠️ Project Structure

The project is organized into reusable components for maximum maintainability:

- `src/components/ItemForm.jsx`: Dynamic form for creating and editing items.
- `src/components/ItemList.jsx`: Container for rendering cards.
- `src/components/ItemCard.jsx`: Individual item display with CRUD actions.
- `src/index.css`: Global design system with custom CSS variables.

## 📦 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Locally
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```
